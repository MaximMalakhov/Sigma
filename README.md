# Описание проекта Sigma.

Проект представляет из себя интернет-магазина "Сигма". 
В предоставленных файлах содержится верстка одной из страниц интернет-магазина "Сигма" 
"Оборудование для штрих-кодирования". На этой странице размещены товары данной категории и необходимая информация от товаре.

#Файловая структура
* **css** - каталог содержащий CSS стили проекта. Каталог **css** содержит следующие файлы:
	- **каталог components** содержит файлы отвечающие за основные компоненты страницы. 
		Он включает в себя следующией файлы:
		+ basket.less отвечет за стили корзины;
		+ button.less отвечает за кнопки на странице;
		+ categories.less отвечате за стили категорий сайдбара;
		+ menu.less отвечает за стили всех меню;
		+ pagination.less отвечает за стили навигиции по страницам;
		+ product.less отвечает за стили блока с одним продуктом
		+ products.less отвечат за блок с товарами на странице.
		+ search.less отвечает за стиль блока для поиска по сайту;
		+ services.less отвечает за стили панели сервисов;
		
	- **каталог helpers** содержит файлы упрощающие работу при верстке.
		В нем содержатся следующие файлы:
		+ sprite.less отвечает за спрайт с иконками на странице(создается при использовании команды в gulp).
		+ clearfix.less отвечает за предотвращение схлопываний.
		+ functions.less отвечает за миксины используемые при работе.
		
	- **каталог layout** содержит файлы отвечающие за размеры основных блоков страницы.
		Он включает в себя следующие файлы:
		+ content.less отвечает за стили основного блока с контентом страницы;
		+ footer.less отвечает за стили "подвала";
		+ header.less отвечает за стили header, logo, contacts;
		+ grid.less отвечает за основные блоки страницы.
		+ sidebar.less отвечает за стили блока sidebar;
		+ top-panel отвечает за стили верхней панели страницы;
		+ products.less отвечает за стили блока продуктов;
		
	- **каталог typography**. Содержит следующие файлы:
		+ link.less отвечает за ссылки на странице.
		+ title.less отвечает за заголовки на странице.
		+ list.less стили списков на странице;
		
	- **seting.less** отвечает за общие стили на странице.
	- **style.less** отвечает за сборку основного css файла style.css.
	- **style.css** основной css файл, который собирается из остальных.
		
* **images** - каталог содержащий изображения используемые в проекте. 
	Содержит следующие каталоги:
	- "content" папка с контентным изображение страницы;
	- "decore" папка с декоративными изобрадениями страницы;
	- "icons" папка для спрайтов.
		+ **sprite** каталог, который содержит спрайт(если была использована команда gulp sprite).
* **index.html** - файл содержащий разметку веб-страницы.

##**release** - каталог, который содержит минифицированые файлы.
