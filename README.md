# Описание проекта Sigma.

Проект представляет из себя интернет-магазина "Сигма". 
В предоставленных файлах содержится верстка одной из страниц интернет-магазина "Сигма" 
"Оборудование для штрих-кодирования". На этой странице размещены товары данной категории и необходимая информация от товаре.

#Файловая структура
* **css** - каталог содержащий CSS стили проекта. Каталог **css** содержит следующие файлы:
		
		- **каталог components** содержит файлы отвечающие за основные компоненты страницы. Он включает в себя следующией файлы:
			+ basket.less отвечет за стили корзины;
			+ categories.less отвечает за стили категорий(sidebar)
			+ menu.less отвечает за стили всех меню(top-panel, services, footer);
			+ pagination.less отвечает за стили навигиции по страницам;
			+ product.less отвечает за стили блока с одним продуктом
			+ search.less отвечает за стиль блока для поиска по сайту;
			
		- **каталог helpers**
			+ settings.less отвечает за общие сбросы и стили все страницы, а так же функции и глобальные, необходимые для оптимизации css структуры;
			+ style.less объединяет все less файлы для дальнейшего создания файла style.css
			
		- **каталог layout** содержит файлы отвечающие за размеры основных блоков страницы. Он включает в себя следующие файлы:
			+ content.less отвечает за стили основного блока с контентом страницы;
			+ footer.less отвечает за стили "подвала";
			+ header.less отвечает за стили header, logo, contacts;
			+ main-container овечает за основной размер всего контента на странице;
			+ page.less необходим чтобы зафиксировать "подвал" внизу страницы;
			+products.less отвечает за блок со всемы товарами;
			+ sidebar.less отвечает за стили блока sidebar;
			+top-panel отвечает за стили верхней панели страницы;
			
		- **tкаталог typography**
			+ label.less отвечает за стили надписей на странице;
			+ links.less отвечает за стили ссылок на странице;
			
		- **style.css** основной css файл задающий стили все страницы(собирается из всех less файлов)
* **images** - каталог содержащий изображения используемые в проекте. Содержит следующие каталоги:
	- "content" папка с контентным изображение страницы;
	- "decore" папка с декоративными изобрадениями страницы;
	- "icons" папка для спрайтов;
* **index.html** - файл содержащий разметку веб-страницы.
