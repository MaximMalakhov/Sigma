# Описание проекта Sigma.

Проект представляет из себя интернет-магазина "Сигма". 
В предоставленных файлах содержится верстка одной из страниц интернет-магазина "Сигма" 
"Оборудование для штрих-кодирования". На этой странице размещены товары данной категории и необходимая информация от товаре.

#Файловая структура
* **css** - каталог содержащий CSS стили проекта.В данном файле задаются стили следующих  "блоков":
	- **Верхняя панель** сайта, которая содержит меню и поиск по сайту. Для верстки этих блоков используются следующие селекторы:
		top-panel, menu, search, field и их наследники(если таковые имеются);
	- **Шапка сайта**, которая содержит логотип, контактную информацию, корзину. Для верстки этих блоков используются следующие селекторы:
		logo, contacts, telephone, address, basket, identification, cart и их наследники(если таковые имеются);
	- **Панель сервисов**, содержащая сервисы предостовляемые интернет-магазином. Для верстки этих блоков используются следующие селекторы:
		services, service и их наследники(если таковые имеются);
	- **Боковая панель**, содержащая списки товаров. Для верстки этих блоков используются следующие селекторы:
		catalog, subcategory и их наследники(если таковые имеются);
	- **Каталог товаров**, содержащий основную информацию о товаре и навигацию по сайту. Для верстки этих блоков используются следующие селекторы:
		products, product, pagination,arrow-prev, arrow-next и их наследники(если таковые имеются);
	-**Подвал сайта**, содержащий меню и прочую информацию.  Для верстки этих блоков используются следующие селекторы:
		footer, copyright и их наследники(если таковые имеются);
* **images** - каталог содержащий изображения используемые в проекте. Все изображения помещены в соответствующие папки
	- "images-content";
	- "images-decore";
	- "images-service";
* **index.html** - файл содержащий разметку веб-страницы.
